Platform Features

Built for Doctors & Patients

Role-based access ensures each user sees only what they need

Doctor Dashboard

View assigned patients, write encrypted medical records, manage prescriptions, and handle appointment requests.

Patient Portal

AES-256-GCM Encryption

All medical data — diagnoses, notes, prescriptions — are encrypted at rest. Keys are separate from the database.

AES-256GCMAt-Rest
🪙

JWT API Security

REST API endpoints protected by signed JWT tokens with expiry, refresh tokens, and role-based claims for programmatic access.

RFC 7519BearerRefresh

Secure File Uploads

Patients upload lab reports, scans and documents. Files are renamed, type-validated, size-limited and paths are encrypted.

ValidationRenameEncrypted Path
